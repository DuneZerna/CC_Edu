Let's break it down

  1. When a robot finds the target, a timestamp is set and a message is broadcasted
  2. The recipient of the broadcast differentiates between the time in the message against it's own time
  3. The recipient now compares between all the messages in which it has received, if any, and determines the shortest (newest time) path
